# 2.3 Applications concurrentes ou voisines 

Il est important d'analyser les applications concurrentes ou similaires afin de se positionner stratégiquement sur le marché, d'identifier les bonnes pratiques et d'évaluer ce qui fonctionne moins bien. Pour cela, nous avons exploré différents réseaux sociaux tels que TikTok, YouTube et X, dans le but d'identifier les applications les plus utilisées.

Dans le domaine de l'apprentissage numérique, plusieurs applications et plateformes proposent des solutions similaires ou voisines à Mindlet. Toutefois, aucune d'entre elles ne réunit à la fois l'aspect collaboratif, l'utilisation de l'intelligence artificielle et une interface simple et ludique pour stimuler l'engagement. Voici une analyse des principales applications concurrentes ou voisines :

### 2.3.1 Analyse

On peut constater que de nombreuses applications se rapprochent de Mindlet, mais aucune ne parvient à combiner tous les besoins essentiels : fonctionnalités collaboratives, réseau social, utilisation avancée de l'intelligence artificielle, interface ludique et divers méthodes d'apprentissage.
